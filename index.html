📊 How to Read This Chart
The Punk/BTC Ratio shows how expensive a median non-rare CryptoPunk is relative to Bitcoin.

Percentile: What % of all weeks since Oct 2021 had a lower ratio.

Color Legend:
• ■ 0-20% — Bargain (historically very cheap)
• ■ 20-40% — Modest (below average)
• ■ 40-60% — Fair (around average)
• ■ 60-80% — Pricy (above average)
• ■ 80-100% — Inflated (historically very expensive)

Excludes Aliens, Apes, Zombies, Hoodies, Beanies, 1/7-trait punks, and top 10% rarest.
Data: Etherscan V2 API • Binance API
